Nginx!
Nginx (pronounced "engine X")stylized as NGINX, nginx or NginX, is a web server that can also be used as a reverse proxy, load balancer, mail proxy and HTTP cache. The software was created by Igor Sysoev and publicly released in 2004. Nginx is free and open-source software, released under the terms of the 2-clause BSD license. A large fraction of web servers use NGINX, often as a load balancer.
Working principle of Nginx
The implementation of Nginx will use the Nginx Web Server in conjunction with the Apache Web Server by allowing Nginx to receive a request and send it to Apache to process PHP functionality for display.
Why Nginx ?
- Support for security standards HTTP / 2
- HTTP functionality guaranteed.
- Process static files faster than Apache.
- Supports more users than Apache with asynchronous operation.
